# Dockerfiles
1. Phoenix: Dockerfile for the Elixir [phoenix framework](https://github.com/phoenixframework/phoenix). It install [elixir](http://elixir-lang.org/) v1.0.2 and then gets the Phoenix framework and places it in /Project/phoenix

2. Nerves-SDK: Dockerfile for [nerves-sdk](https://github.com/nerves-project/nerves-sdk) which is used to make firmware with erlang and application as part of the image using buildroot.
